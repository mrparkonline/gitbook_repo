## Vowel Counter

Assume that our vowels in our alphabet are only from this set: `a e i o u`.

Given a string parameter, return the number vowels within it.

```
// Sample Input
Hello, World!

// Sample Output
3
```

There are two vowels in `Hello` and one vowel in `World!`.

* Parameter: `a string`
* Returns: `a number`
